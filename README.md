# Highways 26 - Japanese Carnival Edition

This is the frontend-only implementation of the Highways 26 website, optimized for deployment on Vercel.

## Theme
The theme for Highways 26 is **Japanese Carnival (Matsuri)**. It features vibrant visuals, celebratory elements, and a premium aesthetic that captures the essence of a festive Japanese night.

## Tech Stack
- **Frontend**: React (TypeScript)
- **Styling**: CSS3 (Modern techniques)
- **Deployment**: Vercel (Front-end only)

## Project Structure
- `/src`: React components and logic.
- `/public`: Static assets (images, icons, etc.).
- `vercel.json`: Vercel configuration for SPA routing.

## How to Run Locally
1. Install dependencies: `npm install`
2. Run the development server: `npm start`
3. Open [http://localhost:3000](http://localhost:3000)

## Deployment on Vercel
This project is configured for one-click deployment on Vercel.
The `vercel.json` file handles the client-side routing.
